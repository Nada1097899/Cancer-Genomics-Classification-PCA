# Cancer Classification via Gene Expression Data

A high-dimensional bioinformatics project that classifies various cancer types based on genomic expressions using PCA and Machine Learning.

## Project Overview
This project addresses the challenge of analyzing thousands of gene features to identify specific cancer types. By leveraging dimensionality reduction, the model can accurately distinguish between different tumors (BRCA, COAD, KIRC, LUAD, PRAD).

## Tech Stack
* **Language:** Python
* **Machine Learning:** Support Vector Machine (SVM)
* **Dimensionality Reduction:** Principal Component Analysis (PCA)
* **Libraries:** Pandas, Scikit-learn, Seaborn, Matplotlib.

## Key Technical Steps
- **Handling High Dimensionality:** Processed a dataset with 20,531 gene features.
- **PCA Implementation:** Reduced features while preserving variance to improve model efficiency and prevent overfitting.
- **Model Evaluation:** Achieved an outstanding accuracy of **98.7%** using SVM.

##  Results & Performance
- **Accuracy:** 98.7%
- **Metrics:** High F1-scores across all 5 cancer categories, demonstrating the robustness of the model.
- **Visualizations:** Included PCA scatter plots showing clear separation between cancer types.
